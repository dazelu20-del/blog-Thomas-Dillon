# Vibe Journal

A responsive static blog website with a polished editorial homepage, topic cards,
searchable article previews, and a newsletter signup interaction.

## Open the site

[Open the blog website](https://dazelu20-del.github.io/vibe/)

The link will be live after the GitHub Pages workflow runs from `main`.

## Files

- `index.html` - Page structure and sample blog content.
- `styles.css` - Responsive visual design and layout.
- `script.js` - Mobile navigation, post filtering, and newsletter form behavior.

## Run locally

Serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
